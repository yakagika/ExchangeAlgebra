class (HatVal n, HatBaseClass b, Monoid (a n b)) => Redundant a n b where Source #

Type class for Redundant Algebra. Provides fundamental exchange algebra operations: hat, bar, norm, scalar product, and compress.

Redundant ⊃ Exchange

hat calculation >>> (.^) (10:Not:<Cash .+ 10:Hat:<Deposits) 10.00:Hat:<Cash .+ 10.00:Not:<Deposits

bar calculation >>> x = 10:Not:<Cash .+ 10:Hat:<Deposits >>> y = 5:Hat:<Cash .+ 5:Not:<Deposits >>> (.-) $ x .+ y 5.00:Not:<Cash .+ 5.00:Hat:<Deposits

norm calculation >>> norm $ 10:Not:<Cash .+ 10:Hat:<Deposits 20.0

(.*) calculation >>> (.*) 5 $ 10:Not:<Cash .+ 10:Hat:<Deposits 50.00:Not:<Cash .+ 50.00:Hat:<Deposits

compress calculation >>> compress $ 10:Not:<Cash .+ 5:Hat:<Cash .+ 3:Not:<Cash 5.00:Hat:<Cash .+ 13.00:@Not:<Cash

Minimal complete definition

(.^), (.-), compress, (.+), (.*), norm

(.^) :: a n b -> a n b infixr 7 Source #

Hat operation. Flips Hat/Not on all elements. Complexity: O(1) for singleton, O(n) for Liner (n is the number of base keys)

(.-) :: a n b -> a n b infixr 2 Source #

Bar operation. Cancels Hat/Not on the same base and retains only the difference. Complexity: O(n) (n is the number of base keys)

bar :: a n b -> a n b Source #

Alias for bar operation. Identical to (.-).

compress :: a n b -> a n b Source #

Aggregates values on the same base. Sums while preserving the Hat/Not distinction. Complexity: O(n) (n is the number of base keys)

(.+) :: a n b -> a n b -> a n b infixr 3 Source #

Addition of algebra elements. Alias for the Monoid <> operation. Complexity: O(union cost)

(.*) :: n -> a n b -> a n b Source #

Scalar product. Multiplies all element values by a scalar. Complexity: O(1) for singleton, O(n) for Liner

norm :: a n b -> n Source #

Norm. Returns the sum of all element values. Complexity: O(n) (n is the number of base keys)

balanceBy :: (HatVal n, HatBaseClass b) => [b] -> [b] -> Alg n b -> n Source #

Compute the net balance as the difference of two projections. balanceBy plusBases minusBases alg computes projNorm plusBases alg - projNorm minusBases alg.

Useful for calculating stock quantities, profits, etc.

>>> type T = Alg Double (HatBase AccountTitles)
>>> let alg = 100 :@ Not:<Cash .+ 30 :@ Hat:<Cash :: T
>>> balanceBy [Not:<Cash] [Hat:<Cash] alg
70.0

>>> balanceBy [Hat:<Cash] [Not:<Cash] alg
-70.0

foldEntriesToMap :: (HatVal v, HatBaseClass b, Ord k) => (v -> b -> Maybe (k, v)) -> Alg v b -> Map k v Source #

Fold algebra entries into a Map, combining values with (+).

The selector function examines each entry (v, b) and optionally returns a (key, value) pair. Values for duplicate keys are summed.

>>> type T = Alg Double (HatBase AccountTitles)
>>> let alg = 10 :@ Hat:<Cash .+ 20 :@ Hat:<Deposits .+ 5 :@ Hat:<Cash :: T
>>> let f v (Hat :< a) = Just (a, v); f _ _ = Nothing
>>> foldEntriesToMap f alg
fromList [(Cash,15.0),(Deposits,20.0)]
