class (BaseClass a, BaseClass (BasePart a), AxisDecompose (BasePart a)) => HatBaseClass a where Source #

Type class for bases with a Hat component. Provides functionality to decompose and compose a base into its Hat part and BasePart. Manages the credit (Hat) / debit (Not) distinction at the base level in exchange algebra.

Minimal complete definition

base, hat, merge, toHat, toNot, revHat, isHat, isNot
